# 關於 Threadseeker

Hi 感謝你點進這篇文章，我是開發 Threadseeker 的大學生 [Shawn](https://jcshawn.com)，我是一個就讀台灣大學的二年級管學院學生，而 Threadseeker 是一個我單純想做社群分析的專案，希望你喜歡這個我花三天寫出來的小東西。

![image alt text](/images/docs/about/about.jpg)

話說你玩過 [Threadseeker](https://threadseeker.com) 了嗎？如果還沒的話先去玩看看！你可以查詢公開帳號的互動情形跟帳號數據，包含總累積讚數、回覆數等等。

## 開發動機

我自己就是一個常常滑 Threads 的人，而且是會員數三百萬左右就加入了（ 多數人都是一千萬後才加入 ）所以我常在觀察 Threads 社群跟其他平台有哪些不同，包含受眾組成、演算法、未來可能的商業模式等等。

我覺得 Threads 是個很有趣的平台，因為這裡出現爆款貼文的頻率比其他平台高很多，而且平台性質的緣故，很多人都在這裡「 做自己 」，所以引起的討論跟互動都很高。

而且 Instagram 執行長在[某一次的直播時提及台灣是 Threads 表現最好的地區](https://www.instagram.com/p/C4AwD-hvdp3/?hl=zh-tw)，可見台灣人有多愛滑 Threads。

我一直想以數據的角度去理解現在 Threads 的使用現況，所以我就花了 3 天寫一個網站，讓大家可以用數據的方式來看看自己在 Threads 有多活網，以及帳號近期累積的按讚數、回覆數等等。這就是 Threadseeker 的由來。

## 你可以怎麼看數據？

雖然我開發這個網站的一部分主因，是想做成類似 Instagram 心理測驗，那種你會截圖到限動的社群分享性質網站，所以我把這個網站定位成「 活網程度分析器 」而不是社群數據分析工具。

但其實我在設計活網程度計算的計算公式時，其實是用常見的社群經營數據來計算的，主要指標是看「 平均單日的發文數量 」並根據全體資料進行 PR 排序，根據 PR 值給予對應的等級。

當然你也可以用報表的資料簡單計算你的帳號互動率，也就是讚數加上留言數，最後除以總追蹤數，以 Instagram 的標準來說能維持在 1% 以上就很不錯了。

這是我們在經營社群時，用來衡量粉絲對你貼文的互動程度的指標！

所以你可以用 Threadseeker 來看看自己的帳號或知名品牌在 Threads 上的表現，瞭解能引起討論聲量的帳號有哪些共通點，這是我設計這個網站原始的動機。
