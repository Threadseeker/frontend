# よくある質問

このセクションでは、Threadseekerに関する一般的な質問を取り上げています。セキュリティの懸念や統計データの説明についてもここで解答しています。

## なぜGoogleアカウントでログインする必要があるのですか？😖

Googleアカウントでのログインは、バックエンドシステムの不正利用を防ぐためです。これは認証目的のみで使用され、**Googleアカウントのパスワードを取得することはありません。**

Googleを使用してログインすると、サイトは以下の情報を受け取ります：

- アカウント名
- メールアドレス
- プロフィール写真

Threadseekerでは、メールアドレスのみを記録し、以前のログイン履歴や1日のクエリ回数を追跡します（不正利用防止のため）。その他の情報は商業目的やマーケティングには使用しません。

Googleアカウントでのログインが気になる場合は、Threadseekerの利用を控えることもできます。

詳細はこちら：[「Googleアカウントでログイン」機能がデータを安全に共有する方法 - Google.com](https://support.google.com/accounts/answer/12921417?hl=ja)

## なぜGoogleログインのドメインがThreadseekerと異なるのですか？🧐

これはセキュリティの問題ではありません。Threadseekerのバックエンドデータベースサービス（[Supabase](https://supabase.com)を使用）をアップグレードしていないため、Googleログイン画面にSupabaseのデフォルトドメインが表示されます。

これはアカウントのセキュリティには影響しません。ログインページのURLが`google.com`であることを確認してください。Googleアカウントのパスワードにアクセスすることはありません。ランダムに見えるURLはバックエンドデータベースの実際のURLであり、詐欺サイトではありません。

関連情報：[Googleをプロバイダーとして使用する際にcustom.domainに進む - GitHub](https://github.com/orgs/supabase/discussions/5041)

## 分析後、私のThreadsアカウントにリスクはありますか？😯

**結論：ThreadseekerはThreadsアカウントのパスワードやプライバシーデータにアクセスする権限を持っていません。**

サイトは、あなたをフォローしていないThreadsユーザーをプログラムでシミュレートし、プラットフォーム上の公開データと投稿を統計処理して、最終的なレポートに表示します。

そのため、Threadseekerでの分析によってThreadsアカウントがリスクにさらされることはありません。サイトのレポートに表示されるデータは、誰でも見ることができる情報です。

基本的に、プログラムが取得できるデータはGoogleが取得できるデータと同じで、プログラミングを使用してより効率的に統計を取っているだけです。

（ご理解いただきたいのは、誰でもプログラムを書いてThreads、Instagram、FacebookなどのSNSの公開データを自動的に分析できるということです。分析を避けるには、アカウントを非公開に設定するしかありません）

## 私のThreadsアカウントの検索履歴は保存されますか？😥

「どのGoogleアカウントがどのThreadsアカウントを検索したか」という検索履歴は、24時間一時的に保存された後に削除されます。

検索履歴を一時的に保存する理由は、現在サイトに1日の検索制限があるためです。各Googleユーザーは1日に1つの新しいThreadsアカウントしか分析できません。Googleアカウントが1日にいくつの新しいThreadsアカウント分析を要求したかを確認するためにこのデータが必要で、マーケティングや統計分析には使用しません。

ただし、すでに分析済みのThreadsアカウントを検索する場合、サイトは検索行為を記録しません。Threadseekerは「未分析のThreadsアカウントの検索」のみを記録します。

同様に、上記の検索履歴が一時的に保存されることが気になる場合は、このサービスを使用しないことをお選びいただけます。

作者のThreadsでの返信：[原文を見る](https://www.threads.net/@cjs.shawn/post/C5wBOjZvcVJ) 👈🏻

## ThreadseekerのGoogleアカウント連携を解除できますか？😗

Threadseekerの使用を中止し、サイトとGoogleアカウントの連携を解除したい場合は、以下のGoogle公式のガイドに従って連携を解除できます：

1. Googleアカウントの[「サードパーティの連携」](https://myaccount.google.com/connections)ページにアクセス
2. 「Googleアカウントでログイン」を選択
3. 連携を解除したいサードパーティアプリやサービスを選択し、「詳細を表示」をクリック
4. 「Googleアカウントでログイン機能を無効にする」を選択し、「確認」をクリック

参考：[Googleアカウントとサードパーティの連携を管理する（Google公式ガイド）](https://support.google.com/accounts/answer/13533235?hl=ja)

## アクティビティレベルはどのように計算されますか？😎

アクティビティレベルは「1日平均投稿数」の指標に基づき、全体データの中でのあなたのアカウントの日々の投稿頻度のPR値に応じて対応するレベルが付与されます。大学入試センター試験の5段階評価のような方式です。

- C: PR 35 以下
- B: PR 35 ~ 40
- B+: PR 40 ~ 50
- A: PR 50 ~ 75
- A+ PR 75 ~ 85
- A++ PR 85 ~ 90
- S: PR 90 - 95
- SSS: PR 95 以上

そのため、分析レポートのアクティビティレベルは時間とデータ量によって変動する可能性があります。
